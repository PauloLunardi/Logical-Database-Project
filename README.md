# Logical-Database-Project

# 📊 Bootcamp de Análise de Dados com SQL DIO

##  Sobre o Projeto

Este repositório contém consultas SQL desenvolvidas em PostgreSQL para análise de dados, com foco em extração de insights, tratamento de informações e apoio à tomada de decisão.

O projeto reúne queries utilizadas em estudos práticos e aplicações reais, organizadas de forma estruturada para facilitar entendimento e reutilização.

---

## 🎯 Objetivo

* Demonstrar habilidades em SQL
* Explorar dados para geração de insights
* Praticar boas práticas de modelagem e consulta
* Construir novo portfólio para oportunidades em Dados / BI

---

## 🧠 Tecnologias Utilizadas

* SQL (PostgreSQL / SQL Server)
* Ferramentas de apoio:

  * Power BI
  * Python (opcional para análise complementar)
