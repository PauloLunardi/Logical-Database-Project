# Projeto Bootcamp DIO - Power Query com Copilot no Excel

## Sobre o Projeto
Este projeto foi desenvolvido durante o **Bootcamp da DIO**, com foco em explorar o uso do **Microsoft Copilot** aliado ao **Power Query no Excel**.  
O objetivo é demonstrar como a inteligência artificial pode apoiar na **transformação, limpeza e análise de dados** de forma prática e eficiente.

## 🚀 Tecnologias Utilizadas
- **Microsoft Excel (365/2019 ou superior)**
- **Power Query (Obter & Transformar)**
- **Microsoft Copilot**
- **Git/GitHub** para versionamento e colaboração

## 🎯 Objetivos
- Aprender a integrar o Copilot ao fluxo de trabalho no Excel.
- Utilizar o Power Query para importar, transformar e preparar dados.
- Criar relatórios automatizados e dinâmicos.
- Documentar o processo de aprendizado e evolução no bootcamp.

## 📂 Estrutura do Projeto
- `/data` → Conjunto de dados utilizados nos exercícios.
- `/scripts` → Exemplos de consultas e transformações no Power Query.
- `/docs` → Documentação complementar e anotações do bootcamp.
- `README.md` → Documento principal com instruções e contexto do projeto.

## 🛠️ Como Executar
1. Abra o Excel e acesse a guia **Dados > Obter Dados**.
2. Carregue os arquivos da pasta `/data`.
3. Utilize os scripts de Power Query disponíveis em `/scripts`.
4. Explore os relatórios e dashboards criados.

## 📖 Aprendizados
- Automação de tarefas repetitivas com Power Query.
- Uso do Copilot para acelerar a criação de consultas e fórmulas.
- Melhoria na organização e análise de dados.
- Integração de IA no fluxo de trabalho de planilhas.
